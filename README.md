# My Claude Skills

Коллекция навыков (skills) для Claude Code и других ИИ-инструментов.

## Навыки

### [ai-engineering-process](ai-engineering-process/)
Инженерный процесс разработки с ИИ: Research → Design → Plan → Implement.
Помогает не терять контроль над кодом при работе с ИИ-агентами.

## Как установить

Скопируй папку нужного навыка в:
```
~/.claude/plugins/marketplaces/claude-plugins-official/plugins/
```
